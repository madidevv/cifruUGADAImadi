# GuessNumber

GuessNumber — простое Android-приложение, разработанное в среде Eclipse (не Android Studio). Пользователю предлагается угадать число, которое случайным образом загадывает приложение.

# Техническое описание

Приложение разработано с использованием JRE 1.8.0, языка программирования Java 7 и Android SDK версии 19.0.0.

# Назначение приложения

Приложение «Угадай число» создано в рамках лабораторной работы по основам разработки мобильных приложений в среде Android IDE и предназначено для изучения структуры Android-проекта и базовых операторов языка Java.

# Описание работы приложения

Программа загадывает случайное число в диапазоне от 0 до 5. Пользователь вводит предполагаемое значение, после чего приложение сообщает результат с помощью подсказок «Много» или «Мало». В случае правильного ответа игра может быть начата заново.

# Функциональные возможности

- Генерация случайного числа с использованием Math.random()

- Обработка пользовательского ввода

- Сравнение введённого и загаданного чисел

- Сброс состояния и запуск новой игры

# Структура проекта

- src/ — исходный Java-код приложения (MainActivity.java)

- res/layout/ — XML-разметка интерфейса с использованием TableLayout

- res/values/ — строковые и стилевые ресурсы

# Интерфейс приложения

После запуска приложения на экране отображаются текстовое поле с подсказками, поле ввода числа и кнопка управления игрой.
